# Core Service Categories

1. Computing: The virtual engine driving cloud applications
    - Azure Virtual Machines
    - Azure App Service: Manage app and API development in different progrmming languages
    - Azure Functions: VMs
    - Azure Kubernetes Service (AKS)
    - Azure Batch: Big data processor
2. Storage: Valuable data is securely stored
    - Azure Blob Storage: Unstructured data storage, cheaper
    - Azure SQL Database
    - Azure Files: Network files
    - Azure Disk Storage: Persistent data VM storage
    - Azure Data Lake Storage (big data)
3. Networking: Enables seamless communication and data flow in the cloud
    - Azure Virtual Network
    - Azure Content Delivery Network
    - Azure Load Balancer
    - Azure ExpressRoute: Private connections between data centers and low latency
    - Azure Firewall
