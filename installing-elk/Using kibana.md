# Using Kibana

1. Go to the port where you are running kibana, for this example and configuration, we are running it in the port [5601](http://localhost:5601)
2. Get the enrollment token by running `bin/elasticsearch-create-enrollment-token --scope kibana` where you have runing kibana. For this example, you should run `usr/share/elasticsearch/bin/elasticsearch-create-enrollment-token --scope kibana`
